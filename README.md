# Coffee Rating Application

## Description
A Flask-based Coffee Rating Application where users can vote for different coffee types. Each vote is stored in an SQLite database and displayed dynamically.

## Features
- Display coffee list
- Vote for coffee
- Store votes in SQLite database
- Dynamic vote updates

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Flask
- SQLite

## Run the Project

1. Install Flask

```bash
pip install flask
```

2. Run

```bash
python app.py
```

3. Open

```
http://127.0.0.1:5000
```
